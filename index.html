1. Le choix du support 2. Mettre sur l'écran d'acceuil 3. Changer le nom 4. C'est partie !

1. Le choix du support

App pour téléphone :

Sannez le code QR pour ouvrir le site web sur téléphone
Ouvrir "partager" ios_share

App pour ordinateur :

Cliquez sur le lien plus bas

https://lien/application/

2. Mettre sur l'écran d'acceuil

App pour téléphone :

Cliquez sur "ajouter à la page d'acceuil" add_box

App pour ordinateur :

Ouvrir "partager" ios_share et mettre le site web en "signet" import_contacts

3. Changer le nom

App pour téléphone :

Changer le nom de la page si besoin et cliquez sur ok !

4. C'est partie !
